## Overview
VIDHURA is a web platform aimed at supporting farmers with sustainable farming guidance.
This repository documents the deployment of the VIDHURA static website on AWS using Amazon S3 Static Website Hosting.

## Technologies Used
- HTML, CSS, JavaScript
- Amazon S3
- AWS Bucket Policies
- AWS Academy Sandbox Environment

## Deployment Process
1. Created an Amazon S3 bucket in AWS Sandbox
2. Disabled block public access for website hosting
3. Enabled static website hosting with index.html
4. Uploaded VIDHURA website files
5. Configured bucket policy to allow public read access
6. Verified deployment using the S3 website endpoint
